üìå How GitHub Tokens Work:

What is a Token? A token is a secure credential that acts like a password, allowing the app to authenticate with GitHub on your behalf without storing your actual password.

Creating a Token:
  1. Go to GitHub ‚Üí Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Set Expiration to 90+ days (or custom)
  4. Select scope: repo (full control of private repos)
  5. Click "Generate token" and copy it immediately (you won't see it again)

Security: This token is sent with every request to authenticate you. Never share it or commit it to public repos. If compromised, revoke it immediately in GitHub settings.
